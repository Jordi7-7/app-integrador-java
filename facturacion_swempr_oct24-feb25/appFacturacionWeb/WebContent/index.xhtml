Facturación

Información:

  • Demo realizada con JSF 2.2, Primefaces 6, Eclipse Neon.2, base de datos PostgreSQL 9.2 y EclipseLink 2.5 (JPA).
  • Respaldo de la base de datos facturación en la carpeta bdd del proyecto.
  • Productos gestionados mediante Actions con parámetros, evitando binding con DataTables.
  • Entidades generadas automáticamente; se personalizó la clase FacturaCab para transacciones en cascada.
